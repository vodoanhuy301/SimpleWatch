# SimpleWatch
SimpleWatch with JS, HTML, CSS

- Javascript DOM: 
    + Change img for each colors
    + Change theme: dark and light
- SimpleWatch: 
    + Show hours
    + Show minutes
    + Show seconds
    + Show day
    + Show month
    + Show random heart rate
